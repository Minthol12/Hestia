# Hestia - Predator Case Management & Evidence Automation Framework

**Protect the Vulnerable. Catch the Predators.**

Hestia is a professional-grade case management system designed specifically for investigators tracking online child predators. It streamlines the entire process from initial lead on Doxbin to submission-ready dossier for authorities.

## ⚠️ LEGAL & ETHICAL USE

This tool is designed for **law enforcement, authorized investigators, and victim advocates**. Users are responsible for complying with all applicable laws in their jurisdiction. The developers assume no liability for misuse.

## Features

- **Case Management** - Create and manage multiple investigation cases
- **Doxbin Integration** - Search and extract target data from Doxbin pastes
- **Username Tracking** - Check usernames across 10+ social media platforms
- **IP/Domain Investigation** - Geolocation, WHOIS, and network intelligence
- **Evidence Collection** - Screenshots, metadata extraction, chain of custody
- **Suspect Profiling** - Build comprehensive suspect profiles
- **Timeline Tracking** - Automatic timeline of investigation events
- **Report Generation** - Professional PDF reports with all evidence
- **Evidence Packaging** - ZIP packages with manifests and hashes
- **Submission Forms** - Generate ready-to-submit forms for authorities
- **Anonymity** - Built-in Tor support for safe investigation
